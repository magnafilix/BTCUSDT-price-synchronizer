# Ticker/Coin(btcusdt) Price Synchronizer

## Description

Server app which periodically fetches the BTCUSDT price into database and exposes api with:
- current rate
- exchange rate history

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
